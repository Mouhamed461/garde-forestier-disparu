───────────────────────────────────────────────────────────────
  BUGS CONNUS / POINTS D'ATTENTION
───────────────────────────────────────────────────────────────

  ⚠ Si le serveur redémarre → toutes les sessions sont perdues
    Console et Manette doivent recommencer depuis l'accueil

  ⚠ Si la Manette se déconnecte pendant la vidéo → session supprimée
    Pas de reconnexion implémentée

  ⚠ Navigation automatique Console (3.5s) et Manette (2.5s) :
    si la latence réseau est élevée, les deux écrans peuvent
    se désynchroniser d'une séquence. À surveiller en test WiFi.

  ⚠ Pour déploiement sur réseau local (WiFi) :
    socket.js détecte automatiquement l'IP (window.location.hostname)
    → Console : http://[IP_DU_PC]:3000 (sur le PC)
    → Manette : http://[IP_DU_PC]:3000 (sur l'iPad)
